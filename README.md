# Frendli - Uşaqlar üçün sosial platforma prototipi

Frendli rəngli və təhlükəsiz sosial media təcrübəsi təqdim etmək üçün hazırlanmış interaktiv ön tərəf prototipidir. Layihə sadəcə statik fayllardan ibarətdir və hər hansı backend xidmətinə ehtiyac duymadan brauzerdə işləyir.

## Xüsusiyyətlər
- **Giriş və qeydiyyat:** Brauzer localStorage vasitəsilə uşaqlara uyğun login/register təcrübəsi və şəxsi profil.
- **Profil paneli:** Əhval seçimi, statistikalar və təklif olunan dostlar siyahısı.
- **Post lent:** Mövcud postları görüntülə, yeni post əlavə et və dostlara sürətli şəkildə paylaş.
- **Testlər bölməsi:** Elm, nağıl və riyaziyyat mövzularında qısa viktorinalar.
- **Real-time söhbət:** Anlıq mesajlaşma təcrübəsi üçün simulyasiya edilmiş cavablar və bot bildirişləri.
- **Uşaqlar üçün dizayn:** Parlaq rənglər, yumşaq kölgələr və oynaq animasiyalar.

## Necə istifadə etməli
1. Reponu klonlayın və ya bu qovluğun içərisini yükləyin.
2. `index.html` faylını istənilən müasir brauzerdə açın.
3. Qeydiyyatdan keçərək və ya mövcud istifadəçi ilə daxil olaraq şəxsi profilini aç.
4. Frendli dünyasını kəşf edərək post, test və söhbət funksiyalarını yoxlayın.

## Strukturu
- `index.html` – əsas səhifə və komponentlərin markup-u.
- `styles.css` – vizual dizayn və cavablı şablonlar.
- `app.js` – dinamik funksiyalar, testlər və chat məntiqi.

> Qeyd: Bütün məlumatlar demo məqsədlidir və brauzer yaddaşında saxlanılır.
